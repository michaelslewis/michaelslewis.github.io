Yelp API v2.0

About:
Currently, Yelp offers v2.0 of its API, which allows for searching businesses. While this guide focuses on v2.0 and this is still supported by Yelp at the time of this writing, Yelp is in the process of transitioning to a newer v3.0 API, dubbed "Yelp Fusion". Fusion is beyond the scope of this guide, as it has added requirements for developers in order to access the API, such as submitting the URL of the app under development.
Access:
In order to incorporate the Yelp API into your web application, it is necessary to register to get an authentication key. This process is free and the link for managing your Yelp authentication is here. It will first be necessary to create a Yelp account, which is itself a simple and free process. Once you have signed up, simply return to the above link, and click on the button that reads "Generate new API v2.0 token/secret". This will provide you with a unique "Consumer Key" and "Token", which will be detailed further below. Your unique information will be available whenever you access this page, as long as you are signed into the site. As the site states, it is important that you never share the key or token with anyone. Your authentication allows for a limited number of API calls per day, and anyone who gets your information would be able to use it. Fortunately, if needed, a new key and token can be generated if needed by clicking on that button again.

Unfortunately, the Yelp authentication process is a bit more tricky than that of some other API's. It uses the OAuth 1.0a protocol, which requires several parameters, including the key and token, to be submitted with each request to the API. Not to worry, guidance for making a call will be provided on the next page.
